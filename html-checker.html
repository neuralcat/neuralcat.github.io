HTML Syntax Checker Coursework

This page will check whether a HTML file is correct according to a subset of HTML syntax.

Constraints


  • Only the following HTML tags are supported: html, body, title, h1, h2, h3, p, ul, li, a, div, br, hr
  • A <p> tag cannot be directly placed inside another <p> tag
  • A <div> tag cannot be directly placed inside a <p> tag
  • The <title> tag can only be placed inside a <head> tag
  • There should only be one <head> tag and one <body> tag
  • The whole document should be wrapped in an <html> tag which contains the head followed by the body

Checker

Paste some HTML markup to check, and then press "check HTML."
